# منصة مدكر لإدارة حملة رمضان (نسخة نهائية Static)

منصة تشغيل وإدارة لحملة رمضان الخاصة بـ **جمعية مدكر لتحفيظ القرآن**، مبنية كواجهة Static سريعة قابلة للنشر على Netlify/GitHub.

## ماذا تقدم المنصة؟
- **لوحة رئيسية حية**: تعرض إجمالي تبرعات اليوم، فرص التبرع، عدد السفراء المحققين للمستهدف، وعدد الفروع.
- **صفحة سفير متكاملة**: أهداف يومية، تحديث مباشر للإنجاز، إنشاء فرص تبرع باسم متوفى، كوبونات مكافآت تلقائية، وترتيب تنافسي.
- **صفحة فرع متكاملة**: متابعة أداء سفراء الفرع، مقارنة بالمستهدف اليومي، وكوبون فرع عند تحقيق الهدف.
- **إدارة متكاملة**: فحص سلامة البيانات، محرر JSON (مدخلات/مخرجات)، تطبيق تغييرات محليًا قبل النشر، وتصفير يوم التشغيل.

## التوجيهات التشغيلية
- تستهدف المنصة إدارة أكثر من 800 سفير موزعين على فروع.
- لكل سفير: مستهدف يومي للتبرعات + فتح صناديق.
- لكل فرع: مستهدف يومي للتبرعات + فرص التبرع.
- عند تحقيق الأهداف: يظهر كوبون تلقائي في الصفحة المناسبة.

## المسارات
- `/` الرئيسية (Dashboard)
- `/s/<code>` صفحة السفير
- `/branch/<id>` صفحة الفرع
- `/admin` الإدارة

## البيانات
- `data/content.json` الهوية ووصف الحملة
- `data/branches.json` الفروع ومديريها والمستهدفات
- `data/ambassadors.json` السفراء وربطهم بالفروع والمستهدفات

## النشر
- Build command: فارغ
- Publish directory: `.`
- يدعم SPA عبر ملف `_redirects`.

> ملاحظة: بما أنها نسخة Static فبيانات اليوم (الإنجازات/الكوبونات) تُخزن محليًا عبر LocalStorage، بينما البيانات الهيكلية الأساسية تُدار بملفات JSON.
